P2 - Review Of Linear Algebra 向量与线性代数

向量

向量点乘

  • 快速得到两个向量的夹角：如，光照运算中，法线、光线夹角 / 投影
  • 测量两个向量距离远近
    • 单位向量点乘范围[-1, 1]，1近，-1远
    • 可用于镜面反射，金属高光：反射处附近看，高光明显
  • 判断向量方向，向前/向后
  • 计算

向量叉乘

  • $\vec{x} \times \vec{y} = + \vec{z}$
  • ==判断左右==（叉乘向内/外）==/内外==（三角形三边叉乘同向则在内部，光栅化基础）⭐️

直角坐标系便于向量分解

矩阵

  • 运算法则
    • (AB)C = A(BC)
    • A(B+C) = AB + AC
    • (A+B)C = AC + AB
  • 矩阵转置
  • 单位矩阵
    • 是对角阵
    • 定义矩阵的逆
  • 向量点乘/叉乘写成矩阵形式
